# UI/UX Spec

Design principles:

- Korean labels.
- Calm professional fintech UI.
- No casino-like visuals.
- Strong risk visibility.
- Accessible contrast.
- Responsive by default.

Global status:

- 봇 상태
- 모드
- 실주문 허용 여부
- Worker heartbeat age
- API health
- 오늘 손익
- 오늘 주문 수
- 위험 경고

Navigation:

1. Dashboard
2. Control
3. Watchlist
4. Portfolio
5. Orders
6. Signals
7. Fundamentals
8. News
9. Strategy Lab
10. Settings
11. Logs

Dangerous changes require confirmation. Live trading requires typed Korean phrase.
